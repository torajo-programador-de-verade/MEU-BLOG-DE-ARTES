# Meu Blog Artístico 🎨

Este é o projeto inicial do meu blog pessoal focado em arte, curiosidades e vivências do mundo artístico. Desenvolvido com uma interface minimalista, elegante e com paleta de cores acolhedora destacando o amarelo dourado queimado.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptável para celulares, tablets e computadores.
- **Identidade Visual:** Uso de ícone temático de paleta de cores.
- **Sistema de Interatividade:** Botão de curtida (coração) funcional com contador em tempo real que alterna o estado de clique.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização personalizada, layout responsivo e transições suaves.
- **JavaScript (Vanilla):** Lógica interativa para o botão de curtidas.
- **Font Awesome:** Ícones vetoriais modernos.

## 📂 Como executar o projeto
- Clique no link que está disponível aqui neste projeto, ou:
1. Baixe ou clone este repositório.
2. Certifique-se de manter os arquivos `index.html`, `style.css` e `script.js` na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador de sua preferência.
